# DIY Screen Repair Safety & Tool Checklist (2026)

**Warning:** Physical repair still requires human hands. Agent provides brain + step-by-step guidance only. Risk of further damage is real for beginners.

### Recommended Tools (OEM-quality where possible)
- Heat gun or hairdryer (low setting)
- Precision prying tools + suction cup
- T8000 or B7000 adhesive
- Full screen assembly (not just glass — includes digitizer + flex cables)
- Anti-static tweezers, isopropyl alcohol, microfiber cloth
- Multimeter (optional for testing connections)

### Common Cause for Your Case
Sudden flickering/glitching after long-stable crack is usually:
- Flex cable micro-disconnect or pressure damage
- Progressing micro-fractures in the digitizer
- Loose display connector

Software is ruled out via Safe Mode + restart. If pressure test worsens symptoms → hardware fix needed (full assembly replacement).

### Basic Supervised DIY Flow
1. Backup data first.
2. Power off + disconnect battery if accessible.
3. Heat adhesive around edges.
4. Gently pry and separate old screen.
5. Disconnect flex cables carefully.
6. Install new assembly, reconnect, test before full glue.
7. Calibrate touch/display after reassembly.

Always test connections before final adhesive. If unsure, take to a professional shop.
