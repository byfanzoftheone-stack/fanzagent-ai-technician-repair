# Agent Repair Guide – FanzAgent Ai-Technician Edition

**Goal:** Diagnose and fix sudden glitches (flickering, ghost touch, freezes) on a cracked phone that was stable before.

### Phase 1: Voice Diagnosis (Fully Agent-Orchestrated)
Speak or type:
"Hey FanzAgent, activate ai-technician-repair route. My [phone model] screen was cracked but stable, now suddenly [exact symptoms: flickering lines? worse when pressing edges? freezes?]. Start observer monitoring and diagnose software vs hardware."

Agent team actions:
- Diagnostics Agent: Force restart, Safe Mode, cache clear, pressure test around crack.
- Observer Agent: Watches for patterns during tests.
- If hardware flagged (very common with old cracks) → create diagnosis atom.

### Phase 2: Repair Orchestration
- Research OEM screen assembly + tools.
- Generate custom plan: DIY steps or professional shop coordination.
- Prepare symptom summary for technician.
- Guide data backup if screen is failing.

**DIY Flow (Agent-supervised only):**
- Tools: heat gun/hairdryer, prying tools, suction cup, new full screen assembly.
- Steps: power off, heat adhesive, separate screen, disconnect/reseat flex cables, install new assembly, test & calibrate.

**Warning:** Physical work still needs human hands. Agent provides the brain, planning, and real-time adjustments.

**Professional Route (Recommended):**
Agent finds best local shop, books if possible, and hands off a clean summary.

### Phase 3: Propagation & Compounding
After any repair:
- Run propagation → turn the outcome into a new knowledge atom.
- Example atom: "Sudden flex-cable glitch on cracked [model] after stable period — full assembly replacement fixed it."
- This atom feeds the entire platinum-organism and improves future repairs.

This directly fulfills the original vision: agents that handle "real things" like fixing overnight phone glitches.
